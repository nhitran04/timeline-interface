Introduction to Polaris

With Polaris, a robot reads and performs instructions in steps (i.e., from top to bottom in the numbered containers, as shown below).

Instructions can be in the form of one of the following:

  • An action, which is what the robot does. For example, steps #1 and #3 are actions.
  • A goal, which is what the robot needs to achieve. For example, steps #2 and #4 are goals.

Now that you are familiar with how a robot reads and performs the instructions provided to it in Polaris, let's learn how you can use Polaris to give instructions to the robot to perform.

We'll first learn how to use actions to specify instructions for the robot. After you are comfortable with using actions, we will learn how to use goals.
